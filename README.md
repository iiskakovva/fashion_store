# Fashion Store - Магазин одежды на Django

Современный магазин одежды на Django с управлением пользовательскими настройками через cookies.

## Возможности

- 🛍️ Просмотр товаров по стилям (Повседневный, Формальный, Спортивный, Винтаж, Бохо)
- ⚙️ Настройка любимых стилей и предпочтительных размеров
- 🌙 Выбор светлой/темной темы
- 🌍 Настройка языка интерфейса
- 📚 История просмотренных товаров
- 📱 Адаптивный дизайн
- 🍪 Сохранение настроек в cookies браузера
